# AtCoder Problem Converter

Convert AtCoder problem HTML pages to clean Markdown.

### Usage

```bash
uvx --from git+https://github.com/ichyo/atcoder-problem-converter.git apc https://atcoder.jp/contests/abc419/tasks/abc419_a
```

or

```bash
$ uv tool install git+https://github.com/ichyo/atcoder-problem-converter.git
$ apc https://atcoder.jp/contests/abc419/tasks/abc419_a
```
